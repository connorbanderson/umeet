# ✨ uMeet ✨

### Motivation
A meeting app where users can sign up and create local events. Used to practice my Vue JS skills<br>

### Technologies Used
* Vue Ant Design
* Firebase
* Self Made SVG's in Adobe Illustrator
* Animated SVGs

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
